## Google Apps Script Web App Notes

From the official Google Apps Script web app guide, the waitlist integration setup in the README should instruct the user to deploy the script using **Deploy > New deployment**, enable the **Web app** deployment type, complete the deployment configuration, and then click **Deploy**. The guide also notes that the deployed URL can be shared with users who have been granted access, and that execution/access settings must be chosen appropriately for the intended audience.

> "To deploy a script as a web app, follow these steps: 1. At the top right of the script project, click Deploy > New deployment. 2. Next to \"Select type,\" click Enable deployment types > Web app. 3. Enter the information about your web app in the fields under \"Deployment configuration.\" 4. Click Deploy." — Google for Developers, Apps Script Web Apps guide
